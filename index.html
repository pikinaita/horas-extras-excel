üöÄ Instalacion y uso

  1. Descarga el ZIP y extraelo en una carpeta de tu eleccion
  2. Abre una terminal en esa carpeta e instala dependencias:
    # Instalar dependencias
    pip install -r requirements.txt
  3. Ejecuta la aplicacion:
    python main.py
  4. La primera vez se abrira el explorador de archivos: selecciona tu parte mensual Excel
  5. Indica el dia y el mes que quieres registrar
  6. Elige el turno: Manana Tarde Noche
  7. Indica si es Sustitucion Operador o Sustitucion Jefe de Turno
  8. La app marca la X en la celda correcta y guarda el archivo conservando todo el formato

‚ÑπÔ∏è Como funciona

  1. Busca la hoja del mes por prefijo (ej: Enero, Feb)
  2. Localiza la fila por el dia en la columna A (compatible con formulas =DIA())
  3. Columna C = Manana   D = Tarde   E = Noche
  4. Columnas P:Q:R:S (combinadas) = Tipo de sustitucion
  5. Preserva 100% el formato original del Excel (estilos, formulas, celdas combinadas)
